# Module 4 - TDD using JUnit5 and Mockito

Covers test-driven development using JUnit 5, Mockito for mocking/stubbing, SLF4J logging, and Lombok.

## Key Topics
- JUnit 5 Setup, Assertions, AAA Pattern
- Mockito: Mocking, Stubbing, Verification
- SLF4J Logging
- Project Lombok
