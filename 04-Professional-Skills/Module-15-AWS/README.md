# Module 15 - AWS Cloud Fundamentals

> **Note:** This module is for personal learning only. No hands-on exercises are assigned.

Covers AWS cloud services including IAM, EC2, S3, RDS, DynamoDB, Lambda, and API Gateway.

## Key Topics
- AWS IAM (Users, Roles, Policies)
- EC2 and S3
- RDS and DynamoDB
- AWS Lambda (Serverless)
- AWS API Gateway
